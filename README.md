# Open Cashback

## Stack
- nestjs
- node v14 - v16
- mongodb

## Run
### With Docker Compose
```$ docker-compose up```

### Without Docker
_run mongodb before this command_
```$ npm run start:dev```

## Tests
```$ npm run test```
